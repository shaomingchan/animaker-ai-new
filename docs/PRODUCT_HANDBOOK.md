## 一、产品定义

**Animaker.AI** 是一个 AI 驱动的动作迁移工具，核心功能是：
- 用户上传一张人物照片
- 选择一个参考动作视频（或从预设库选择）
- AI 自动将视频中的动作迁移到照片中的人物，生成舞蹈/动作视频

**一句话描述：** Make Any Photo Dance — 让任何照片里的人动起来

**官网：** https://animaker.dev（原 animaker-ai.vercel.app 已重定向）

---

## 二、核心卖点（必须准确传达）

| 卖点 | 说明 | ❌ 常见错误 |
|------|------|-----------|
| **无需 prompt** | 用户不需要写任何文字描述，只需上传照片+选视频 | 说"用 prompt 生成视频" |
| **无需编辑技能** | 完全自动化，不需要剪辑/PS/AE 等技能 | 说"编辑你的视频" |
| **三步完成** | 上传照片 → 选动作 → 下载视频 | 描述成复杂流程 |
| **5-15 分钟出片** | 大多数视频 5-15 分钟生成完毕 | 说"秒出"或"实时" |
| **动作迁移** | 是把参考视频的动作迁移到照片人物上，不是"AI 生成视频" | 说"文生视频"或"AI 创作" |

## 三、用户旅程（精确流程）

```
1. 访问 animaker.dev
2. 点击 "Try It Now — $1.99" 或 "Get Started"
3. 上传一张人物照片（JPG/PNG/WebP，需清晰显示人物）
4. 上传参考动作视频 或 从预设动作库中选择
5. 等待 AI 生成（5-15 分钟）
6. 预览 → 下载视频
7. 可用于任何用途（包括商业用途）
```

**关键细节：**
- 照片类型：自拍、肖像、全身照均可
- 视频长度：最长 30 秒
- 分辨率：单次购买 540P，10-Pack 支持全分辨率
- 没有订阅制，按次付费或买积分包

## 七、禁止用语清单

以下表述在任何输出中都不允许出现：

| ❌ 禁止 | ✅ 正确 | 原因 |
|---------|--------|------|
| "用这个 prompt 生成视频" | "上传照片，选个动作" | 产品不需要 prompt |
| "AI 视频生成器" | "AI 动作迁移工具" | 我们不是通用视频生成 |
| "编辑你的视频" | "下载你的视频" | 用户不需要编辑 |
| "秒出"/"实时生成" | "5-15 分钟生成" | 不要夸大速度 |
| "免费试用" | "$1.99 起" | 没有免费方案 |
| "订阅制" | "按次付费" | 没有订阅 |
